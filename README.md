#  Hand Gesture LED & Buzzer Control

Real-time computer vision system that controls multiple LEDs and a buzzer using hand gestures detected via webcam.

##  Features
- Hand tracking with MediaPipe
- LED control via hand gestures
- Buzzer notes control
- Serial communication with Arduino
- Real-time feedback with on-screen legend
